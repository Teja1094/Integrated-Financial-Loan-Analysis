# Integrated-Financial-Loan-Analysis
Project Overview 

The Bank Loan Analysis Dashboard is an interactive Power BI solution developed to analyze loan application trends, loan performance, borrower characteristics, and lending patterns. 

The dashboard provides business users with actionable insights into good and bad loans, regional lending distribution, loan purposes, employment profiles, and home ownership categories.

Project Objectives

Analyze overall loan portfolio performance.
Identify Good Loans and Bad Loans based on loan status.
Monitor monthly loan application and funding trends.
Evaluate regional loan distribution across states.
Analyze loan applications by employment length and loan term.
Identify major borrowing reasons through loan purpose analysis.
Compare lending patterns across home ownership categories.
Enable interactive navigation using bookmarks and action buttons.

Business Benefits

Enables monitoring of loan portfolio health.
Helps identify high-risk loan segments.
Supports data-driven lending decisions.
Improves visibility into borrower behavior.
Provides quick access to key performance indicators.
Facilitates management reporting through interactive dashboards.

Conclusion

The Bank Loan Analysis Dashboard successfully transforms raw loan data into meaningful business insights. 
Through data cleaning, DAX calculations, interactive visualizations, bookmarks, and navigation features, the solution provides a comprehensive view of loan performance and borrower trends, supporting informed business decision-making.
